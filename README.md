- 开发环境：

	- IDE：Visual Studio 2008 SP1
	- SDK：WinDDK 7600.16385.1
- 调试环境：
	- 虚拟机：VMWare 12
	- 虚拟系统：Windows XP SP3
	- 调试器：Windbg
- 参考书目：
	- 《Windows驱动开发技术详解》——张帆 著，电子工业出版社；
	- 《寒江独钓——Windows内核安全编程》——谭文等 著，电子工业出版社；
	- 《软件调试》——张银奎 著，电子工业出版社；
